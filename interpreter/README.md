# BASIC Interpreter

Classic BASIC interpreter written in Dart.

## Setup

To get started, you'll need to install the Dart SDK. Installation instructions can be found [here](https://dart.dev/get-dart).

When you have the Dart SDK installed, you can get the project running locally by executing the following steps:
1. Clone the repository: `git clone https://github.com/alekshiidenhovi/basic-interpreter.git`
2. Navigate to the project directory: `cd basic-interpreter`
3. Run the tests: `dart test`
